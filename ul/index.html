Unordered Lists (<ul>)

Overview

The <ul> element creates an unordered list of items, typically rendered as a bulleted list. It can contain zero or more <li> elements. Unordered lists are used when the order of the elements is not important to the semantic structure of the page. If the order of the elements does matter/affect the structure of the document, an ordered list (<ol>) should be used instead.

The permitted attributes are the global attributes. The additional attributes compact and type are now considered obsolete and their functionality can be accomplished via CSS.

Uses

Unordered lists should be used when the order of the elements does not impact the structure or meaning of a page. Practical applications for unordered lists include menus or link trees.

CSS Tricks

The most styling on unordered lists comes from styling the bullets (list item markers). As demonstrated above, this should primarily be done using the CSS propert list-style-type. If you want to use a custom bullet, such as an image, the best bet is using the ::before selector on the list items and then setting the background-image. You’d think list-style-image would be the way to go, but it’s extremely limited. For example, you can’t position it or even resize it.
